# 🎬 Movie Library

**Автор:** Колготин Артем Павлович
**Описание:** Графическое приложение (GUI) для ведения личной коллекции фильмов. Позволяет добавлять фильмы, фильтровать их по жанру и году, а также сохранять базу данных в файл JSON.
**Язык:** Python 3.x
**Библиотека:** Tkinter

## ⚙️ Возможности приложения

*   **Добавление фильмов:** Ввод названия, жанра, года выпуска и рейтинга.
*   **Отображение в таблице:** Все добавленные фильмы показываются в удобной таблице.
*   **Фильтрация:** Быстрый поиск фильмов по жанру и/или году выпуска.
*   **Валидация данных:** Проверка, что год — это число, а рейтинг находится в диапазоне от 0 до 10.
*   **Сохранение данных:** Вся коллекция автоматически сохраняется в файл `movies.json` при добавлении нового фильма.

## 🚀 Как запустить проект

1.  **Установите Python** (версии 3.6 или выше) с официального сайта, если он еще не установлен.
2.  **Скачайте или клонируйте** этот репозиторий на свой компьютер.
3.  Откройте терминал (командную строку) в папке проекта.
4.  Запустите приложение командой:
    ```bash
    python main.py
    ```
5.  Готово! Пользуйтесь графическим интерфейсом.

## 📝 Примеры использования

1.  **Добавление фильма:**
    *   Введите *"Начало"* в поле "Название".
    *   Введите *"Фантастика"* в поле "Жанр".
    *   Введите *2010* в поле "Год выпуска".
    *   Введите *8.8* в поле "Рейтинг".
    *   Нажмите кнопку **"Добавить фильм"**. Фильм появится в таблице.

2.  **Фильтрация:**
    *   Чтобы увидеть все фантастические фильмы, введите *"Фантастика"* в поле "Фильтр по жанру" и нажмите **"Применить фильтр"**.
    *   Чтобы найти фильмы за конкретный год, введите *2010* в поле "Фильтр по году".
    *   Можно использовать оба фильтра одновременно для более точного поиска.

3.  **Проверка сохранения:**
    *   Закройте окно программы.
    *   Снова запустите `main.py`.
    *   Все ранее добавленные фильмы снова будут отображаться в таблице.
