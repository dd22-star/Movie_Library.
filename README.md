# 🎬 Movie Library (Личная кинотека)

**Автор:** Колготин Артем Павлович
**Описание:** Графическое приложение (GUI) для ведения личной коллекции фильмов. Позволяет добавлять фильмы с валидацией данных (год/рейтинг), фильтровать их по жанру и году выпуска. Данные сохраняются в локальный файл JSON.
**Язык:** Python 3.x.
**Библиотека интерфейса:** Tkinter.

## ⚙️ Возможности приложения

*   **Добавление фильмов:** Ввод названия, жанра, года выпуска и рейтинга.
*   **Валидация данных:** Проверка на пустые поля; год должен быть числом; рейтинг — числом от 0 до 10.
*   **Отображение в таблице:** Все фильмы выводятся в удобной таблице с прокруткой.
*   **Фильтрация:** Быстрый поиск по жанру и/или году выпуска.
*   **Сохранение данных:** Автоматическая запись и загрузка коллекции из файла `movies.json`.

## 🚀 Инструкция по запуску

1.  Установите Python версии 3.6 или выше.
2.  Склонируйте этот репозиторий или скачайте папку `movie_library`.
3.  Откройте терминал в папке проекта.
4.  Запустите приложение командой:
    ```bash
    python main.py
    ```
5.  Пользуйтесь интерфейсом.

## 📝 Примеры использования

1.  **Добавление фильма:**
    *   Введите *«Начало»* (Название).
    *   Введите *«Фантастика»* (Жанр).
    *   Введите *2010* (Год).
    *   Введите *8.8* (Рейтинг).
    *   Нажмите **«Добавить фильм»**.
2.  **Фильтрация:**
    *   Введите *«Комедия»* в поле «Фильтр по жанру» и нажмите **«Применить фильтр»**.
    *   Введите *2023* в поле «Фильтр по году».
3.  **Проверка сохранения:**
    *   Закройте программу.
    *   Запустите её снова (`python main.py`).
    *   Все ранее добавленные фильмы будут на месте.
